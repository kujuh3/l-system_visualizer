# l-system_visualizer
L-System visualizer made with python and turtle.

<i>In progress.</i>

Example: Sierpinski triangle using a Sierpiński arrowhead curve L-system:
   - variables : A B
   - constants : + −
   - start  : A
   - rules  : (A → B−A−B), (B → A+B+A)
   - angle  : 60°
